1.新建动态Web工程：

    编码设置为“utf-8”，Jre设置为默认jre

2.添加框架环境：

    Junit:   
     1.右键BuildPAth---AddLibraries 
  
    Struts2： 
     1.加入jar包
     2.添加配置文件struts.xml
     3.在web.xml中添加过滤器(Filter)

    Hibernate:
     1.加入jar包
     2.添加Hibernate配置文件(hibernate.cfg.xml)
   
    Spring:
     1.加入jar包
     2.添加Spring配置文件(applicationContext.xml)

3.整合框架：

    Struts2与Spring整合：
     1.添加Struts2与Spring的插件jar包
     2.在web.xml中添加监听器(Listener)
     3.在 struts.xml中配置 Action 时, class 属性指向该 Action 在 IOC 中的 id

    Hibernate与Spring整合：
     1.添加c3p0和MySQL的驱动jar包
     2.新建db.properties（配置基本属性、c3p0连接池）
     3.hibernate.cfg.xml配置:数据源(导入db.properties)、SessionFactory、声明式事务

4.环境测试：

    1.资源分类：action包、service包、dao包、config包
    2.配置一个实体类/映射文件，映射文件放入hibernate.cfg.xml中
    3.编写action、service、基础dao、jsp，配置struts.xml
